# SmartSpend v2 — Страницы приложения

## Публичные страницы

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/` | `Landing.jsx` | Лендинг с описанием продукта и квиз-модалом для онбординга. После прохождения квиза записывает `ss_auth='true'` и перенаправляет в ленту. |

---

## Основные страницы приложения

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/feed` | `Feed.jsx` | Лента материалов — статьи, наборы, рекомендации. Главная страница после авторизации. |
| `/catalog` | `Catalog.jsx` | Каталог наборов вещей. Поиск, фильтрация по категориям, просмотр карточек наборов. |
| `/inventory` | `Inventory.jsx` | Инвентарь пользователя. Группы вещей с ring-прогрессом по статусам (ok / soon / urgent / overexploit). |
| `/profile` | `Profile.jsx` | Профиль пользователя с табами: наборы, статьи, избранное. Hero-карточка с аватаром и статистикой. |
| `/settings` | `Settings.jsx` | Настройки приложения — тема, язык, уведомления и прочие параметры. |
| `/account` | `Account.jsx` | Управление аккаунтом — имя пользователя, email, безопасность. |

---

## Детальные страницы

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/set/:id` | `SetDetail.jsx` | Детальная страница набора вещей. Hero-карточка, список предметов, теги, действия. |
| `/article/:id` | `Article.jsx` | Страница статьи с hero-блоком, форматированным контентом и метаданными автора. |

---

## Страницы создания

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/create-set` | `CreateSet.jsx` | Форма создания нового набора. Выбор типа, категории (чипы), добавление вещей. |
| `/create-article` | `CreateArticle.jsx` | Редактор статей с toolbar для форматирования текста (жирный, курсив, заголовки и др.). |

---

## Системные страницы

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/notifications` | `Notifications.jsx` | Список уведомлений пользователя с группировкой по дате. |
